Название книги: Додо книга
Общая оценка: 3,7 / 5
Оценка автора: 2 / 5
Отзыв: Книга предназначена для сотрудников компании Додо. По идее должна мотивиррвать на свершения. Каких-то уникальных мыслей не нашел. Либо они есть, но для меня это банальность. Есть парочку интересных историй. По степени бесполезности могу сравнить с книгой "Дело не в кофе".
Ожидал больше крактических советов.
