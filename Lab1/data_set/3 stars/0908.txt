Название книги: Детские травмы больших взрослых. Как преодолеть то, что родом из детства
Общая оценка: 2,6 / 5
Оценка автора: 3 / 5
Отзыв: Книга для тех, кто только начинает знакомиться с психологией. Вводная, обзорная экскурсия по психологическим травмам. Отсутствует чёткая структура, присутствует общая неравномерность повествования, лишённая логических переходов. Научно-популярная психология для несведущих.
