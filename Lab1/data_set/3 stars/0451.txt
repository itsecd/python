Название книги: СУБД для программиста. Базы данных изнутри
Общая оценка: 4,7 / 5
Оценка автора: 3 / 5
Отзыв: Отформатировано странно, для кого написано непонятно, темы подобраны в лучших традициях классификатора Борхеса. При этом по тексту разбросаны довольно едкие (и небезынтересные) реплики автора обо всём подряд.
Книга не вот чтобы плохая, но она, скорее, подбрасывает какие-то темы для самостоятельного изучения, чем помогает. Даже как учебное пособие для вразумления начинающих не очень годится -- в отличие от какого-нибудь увесистого тома про устройство этих ваших систем хранения табличков -- лёгкая слишком.
